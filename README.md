New Portfolio

About
This is my personal portfolio website, built with HTML and CSS. The portfolio showcases my projects, skills, and experiences, and serves as an online resume for potential employers and collaborators.

Features
Responsive Design: Optimized for all screen sizes.
Custom CSS Styling: Unique and clean design tailored to reflect my personal style.
Interactive Elements: Includes animations and hover effects using CSS.
Projects Showcase: Displays my best work with detailed descriptions and links.
